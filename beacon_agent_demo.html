Beacon

An agent that finds, ranks, translates, and explains opportunities — for students anywhere the listings don't reach.

Hackathons, internships, and youth programs exist all over the world — but most listings are posted in one or two dominant languages, on platforms that skew toward students already in well-connected cities and institutions. Students in rural districts, smaller towns, and underserved regions of any country are routinely left out, not for lack of ability, but for lack of someone translating and filtering on their behalf. Beacon is an agent that does that work automatically.

How this demo works: this is a rule-based simulation of the agent pipeline for demo purposes — the filtering, ranking, and "why this matches you" reasoning run on a small local logic layer instead of a live LLM call, and translated text is pulled from a pre-built table rather than a live translation API. In the production build, the same four steps (Fetch → Filter & Rank → Translate & Simplify → Explain) are handled by an LLM agent that pulls live listings from global and regional opportunity platforms, reasons over each student's profile to score relevance, and generates the plain-language explanation per result in their language.
