Beacon
An opportunity agent for students
anywhere the listings don't reach

Hackathons, internships, and youth programs exist globally — but they're listed in one or two dominant languages, on platforms that weren't built for students in smaller cities or rural districts. Beacon is an agent that fetches those listings, filters and ranks them against your profile, translates them, and explains why each one fits you specifically.

API Key

This demo calls Gemini live for all filtering, ranking, translation, and explanation — the reasoning is a real model, not a script. Paste your own key to activate it (free at aistudio.google.com/apikey). The key stays in your browser and is sent only directly to Google's API — never to any other server. Leave blank to see the simulated version instead.
